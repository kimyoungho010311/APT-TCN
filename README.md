# 🏢 아파트 가격 예측: 멀티모달 시계열 및 시나리오 분석

## 1. 연구 주제 선정 이유

- 문제 인식: 대한민국의 가계 자산의 약 75.2%가 부동산에 편중되어 있어, 정확한 부동산 가격 예측은 개인의 자산 관리 및 경제 안정에 매우 중요합니다.
- 기존 연구의 한계:
  - 대부분의 연구가 단일 시점의 데이터를 사용한 단순 회귀적 설명에 머물러 자산 증식에 실질적인 도움을 주지 못했습니다.
  - 시계열 예측을 수행한 경우에도 한 가지의 미래만 예측하는 결정론적 예측에 머물러 금리 변동이나 정책 변화와 같은 미래 불확실성을 반영하지 못했습니다.
- 본 연구의 목적: 아파트 단지 정보, 거시 경제지표, 위성 이미지와 같은 멀티모달 데이터를 결합하고, 시나리오 분석을 적용하여 미래 경제 상황 변화에 따른 가격 변동 범위를 낙관적/중립적/보수적 관점에서 예측하는 종랍 적인 모델을 제시하는 것입니다.

## 2. 💾데이터 구성 및 전처리

### 2.1 데이터 전처리
- 데이터 분할: 데이터 누수를 방지하기 위해 전체 데이터를 **시간순**으로 훈련(7):검증(2):테스트(1) 비율로 분할했습니다.
- 타겟 변수 변환: 타겟변수인 `면적당 단가(만원)`은 이상치에 치우친 분포를 보였기 떄문에, 학습 안정성을 위해 **로그 변환**을 적용했습니다.
- 스케일링 기법:
  - `MinMaxScaler`,`StandardScaler`,`RobustScaler`를 비교한 결과, **MinMaxScaler**가 모든 예측 구간에서 가장 낮은 MAE를 기록했습니다(아래 표 참조).

| Scaling Method | 1개월 후 | 3개월 후 | 6개월 후 |
|---------------|----------|----------|----------|
| MinMax        | **7.107**|**6.991** |**7.012** |
| Standard      | 7.182    | 7.254    | 7.201    |
| Robust        | 7.165    | 7.231    | 7.182    |

- 이상치 처리: 아래 그림에 나와있다시피 이상치가 다수 존재하는 모습을 확인하였으나, 이러한 데이터들은 강남의 초고가 매물을 표현하는 데이터라고 판단하여 이상치에 대한 전처리는 수행하지 않았습니다.

## 💻 3. 시스템 아키텍처 및 실험

### 3.1 멀티모달 특징 융합

<img width="881" height="342" alt="image" src="https://github.com/user-attachments/assets/582ef88b-dd15-45c8-a6b9-efced2bb12cb" />

1. 정형 특징 추출: 정형 데이터(아파트 거래 + 경제 지표)는 **MLP**를 통해 64차원 벡터로 변환됩니다.
2. 비정형 특징 추출: 비정형 데이터(위성 이미지)는 **ResNet-50**을 통과한 후, 차원의 저주 문제 해결을 위해 추가적인 MLP 레이어를 거쳐 32차원의 벡터로 변환됩니다.
3. 특징 융합: 64차원 정형 벡터와 32차원 비정형 벡터를 **병합**한 후, 다시 중앙의 **MLP**를 통과시켜 두 이질적인 데이터 간의 **복잡한 비선형 상호작용**을 학습합니다. 최종적으로 256차원의 **멀티모달 데이터셋**을 구축합니다.

### 3.2 실험 및 결과

- 평가 지표: **MAE(Mean Absolute Error)**를 사용했습니다. 타겟 변수에 이상치가 존재하므로, 오차를 제곱하지 않아 이상치에 덜 민감한 MAE가 모델의 일반적인 예측 성능을 안정적으로 평가하기에 적합하다고 판단했습니다.
