# 🏘️ 시나리오에 따른 멀티 모달 데이터 기반 아파트 가격 예측

본 저장소는 [시나리오에 따른 멀티 모달 데이터 기반 아파트 가격 예측]의 공식 구현 코드입니다.

#### 1. 연구 요약 (Abstract)

부동산 시장은 복잡한 비선형성과 높은 변동성을 특징으로 합니다. 기존 연구는 주로 실거래가와 같은 정형 데이터에 의존했으나, 이는 아파트의 입지, 환경 등 비정형적인 공간 정보를 반영하는 데 한계가 있었습니다. 또한, 기존의 결정론적 예측(Deterministic forecasting)은 미래의 금리 변동이나 정책 변화와 같은 불확실성을 반영하지 못한다는 단점이 있었습니다.

본 연구는 이러한 한계를 극복하기 위해 다음과 같은 접근 방식을 제안합니다.
- 멀티모달 융합: 정형 데이터(실거래가, 거시 경제지표)와 비정형 데이터(위성 이미지)를 융합하여 예측의 정확성을 높입니다.
- 시나리오 기반 예측: 미래 거시 경제 변화를 가정한 3가지 시나리오(낙관, 중립, 보수)를 설정하여 불확실한 미래에 대한 다양한 예측 경로를 제시합니다.

### 3. 실험 및 결과
#### 3.1 위성 이미지 효과성 검증
멀티모달 데이터셋 구축을 위해 위성 이미지가 예측 성능에 미치는 영향을 검증하기 위해, 베이스 모델(정형 데이터만 사용)과 제안 모델(위성 이미지 융합)의 성능을 비교하였습니다.  

|모델 구분| 1개월 후 MAE| 3개월 후 MAE| 6개월 후 MAE|
|-------|-----------|-----------|------------|
|Baseline model|7.486|     7.487|       7.236|
|Proposed model| 7.137|    7.212|       7.156|
|감소율   |     4.66%|       3.67%|       1.11%|

실험 결과, 위성 이미지를 융합했을 떄 모든 예측 구간에서 MAE가 감소하여 예측 정확도가 향상됨을 확인했습니다.

#### 3.2 최종 예측 모델 선정
GRU, LSTM, TCN 세 가지 시계열 모델의 성능을 비교한 결과, TCN 모델이 모든 구간에서 가장 우수한 성능을 보여 최종 예측 모델로 선정되었습니다.  

|예측 시점| GRU   | LSTM | TCN|
|-------|-------|------|----|
|1개월 후 |7.305  |7.237 |**7.219**|
|3개월 후|7.457|7.212|**7.210**|
|6개월 후|7.527|7.256|**7.242**|

#### 3.3 시나리오 기반 미래 예측
선정된 TCN 모델을 사용하여 미래 12개월동안의 아파트 가격을 3가지 시나리오에 따라 예측했습니다.
- 시나리오 설정: 다양한 경제지표의 월별 변화율을 다르게 적용 (상세 변화율은 논문 참조)
